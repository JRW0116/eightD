# eightD

Static GitHub Pages app for filling out an 8D Problem Solving Report.

## Features

- Mirrors the provided 8D Word template as a live browser form
- Saves drafts in the browser automatically
- Exports and imports report data as JSON
- Supports print or PDF output from the browser
- Includes editable row-based tables for team members, analysis, approvals, and action tracking

## Publish with GitHub Pages

1. Push these files to the `main` branch of `JRW0116/eightD`.
2. In GitHub, open `Settings` > `Pages`.
3. Set the source to `Deploy from a branch`.
4. Choose the `main` branch and the `/ (root)` folder.
5. Save the settings.

GitHub Pages will then host `index.html` as the app entry point.
